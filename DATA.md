# Data and Artifact Statement

This repository includes the processed FuseEval-lite subset used for the reported reproduction and the final model outputs generated during the experiment. The subset contains 400 examples: 200 English and 200 Chinese.

The repository does not include model checkpoints, Hugging Face cache files, authentication credentials, API keys, account names, passwords, or failed intermediate runs.

Users who regenerate model outputs are responsible for complying with the licenses and access terms of the referenced upstream model and dataset providers.
